### packing and unpacking
#### packing

```python
# list
old_list = [1,2,3,4]
new_list = [i+1 for i in old_list ]
# dict
old_dict = {‘key1’:1,’key2’:2,”key3”:3}
new_dict = {k:v+1 for k,v in old_dict.iteritems() }
```

#### unpacking

```python
#swap
```
